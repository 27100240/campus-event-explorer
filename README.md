Campus & Community Event Explorer

A clean, user-focused event discovery web application designed to help students and communities easily browse, filter, and save events that matter to them.

Overview

Campus event information is often scattered across WhatsApp groups, Instagram posts, and spreadsheets, making it difficult for users to stay updated.

This project reimagines event discovery as a simple, structured, and accessible interface, where users can:

search events
filter by category
sort by date
save events for later

The goal is to prioritize usability, clarity, and discoverability over complex operational features.

Features:
- Search Events
Find events by keywords, venue, or description
- Category Filters
Filter events by type (Talks, Workshops, Career, Culture, etc.)
- Sort by Date
View events in chronological order (earliest first)
- Save / Unsave Events
Bookmark events and maintain a personal shortlist
- Event Details
Clean presentation of event information including time, location, and description
- Responsive UI
Designed for both desktop and mobile usability

Tech Stack:
- React (Vite)
- JavaScript (ES6)
- CSS (custom styling)
- Lucide React (icons)

Why I Built This


This project was inspired by my course work on a campus event platform and the challenges students face in navigating fragmented event information.



Through this project, I focused on:
- improving event discoverability
- designing a clean and intuitive UI
- building a system that prioritizes user experience over complexity


It also reflects my broader interest in human-centered design and accessible technology, especially in community-driven environments.

Future Improvements:
- Add event detail pages
- Integrate real-time event data
- Add location-based filtering or map view
- Enable event submission flow
- Add calendar integration
